C#Property-to-Mvvm

Convert C#-Property to C#-Mvvm-Property

This tool instantly converts C#-Property into a C#-Mvvm-Property type definition. Paste a C#-Property structure on the left and the equivalent C#-Mvvm-Property type will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output!
